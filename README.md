# LimbusDialogueBoxes_pt-br

Tradução em **Português (PT-BR)** das caixas de diálogo de batalha de **Limbus Company**.

## Visão geral

Este projeto foi criado para traduzir para PT-BR os diálogos exibidos nas **speech bubbles** durante as batalhas de **Limbus Company**.

A base utilizada foi o arquivo `BattleSpeechBubbleDlg` e `BattleSpeechBubbleDlg_mowe` disponibilizado no projeto **LimbusDialogueBoxes_EN**, de **NotherWael**, que organiza e traduz essas falas para o inglês.

A partir desse material, foi desenvolvido um processo de conversão **EN → PT-BR** para preencher essa lacuna e ajudar a comunidade brasileira.

---

## O que este projeto faz

- Traduz os campos `dlg` do arquivo JSON
- Mantém intactos os campos `id`, `desc` e a estrutura do arquivo
- Preserva tags e marcações especiais
- Gera uma versão em PT-BR pronta para revisão e uso
